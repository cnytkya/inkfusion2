SECURITY FEATURES IMPLEMENTED
================================================================================

✓ Session-Based Authentication
  - Secure session storage in application memory
  - 30-minute idle timeout
  - Session key: UserEmail

✓ Secure Cookies
  - HttpOnly flag (prevents JavaScript access to session cookie)
  - SameSite=Lax (prevents CSRF attacks)
  - Secure flag for HTTPS connections

✓ CSRF Protection
  - Anti-forgery token validation on login form
  - Token included in all POST requests

✓ Password Security
  - Constant-time comparison (prevents timing attacks)
  - Credentials stored as constants (for now)

✓ Access Control
  - [RequireLogin] attribute protects admin pages
  - Automatic redirect to login for unauthenticated users
  - Session validation on every request

✓ Logging
  - Login attempts logged with timestamps
  - Failed login attempts recorded
  - Logout actions tracked
